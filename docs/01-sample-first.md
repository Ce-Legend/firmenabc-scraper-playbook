# 样例先行

这个项目最重要的动作是先跑样例。

企业目录采集看起来很直接，但客户真正关心的是字段是否可用。先跑一个城市/区的小样例，可以提前确认：

- 邮箱字段是否来自页面公开展示。
- 无邮箱企业是否保留。
- 地址是否足够完整。
- Excel / CSV 表头是否符合后续筛选习惯。
- 来源 URL 是否能回查。

## 推荐流程

1. 让客户确认城市/地区 URL。
2. 先跑 1 个地区，限制 10-50 条。
3. 导出 JSONL / CSV。
4. 让客户确认字段和格式。
5. 再按地区拆分全量采集。

## 为什么不一上来全量跑

全量跑完再改字段，成本很高。尤其是邮箱、地址、地区归属、状态说明这些字段，客户确认过以后再长跑会稳很多。
