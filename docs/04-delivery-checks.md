# 交付校验

交付前至少做这几件事。

## 字段检查

- `company_name` 不能为空。
- `source_url` 不能为空。
- 有邮箱时，邮箱格式要通过基础校验。
- `status` 只能是约定值。
- `scraped_at` 要存在。

## 覆盖检查

- 每个目标城市/区都有统计。
- 每个目标城市/区都有导出文件。
- 总表数量和分城市数量能对上。

## 真实性检查

抽样打开 `source_url`，核对：

- 公司名称。
- 邮箱。
- 地址。
- 电话或网站。

## 交付说明

交付时建议附一段说明：

```text
无邮箱企业已保留，email 字段留空。
邮箱只来自 FirmenABC 公开页面。
每条数据都保留 source_url，可回查来源。
失败 URL 已单独记录。
```
